# Autonomous AI Software Engineer

## Overview
Autonomous AI Software Engineer is the default LaunchPulse build path. It is designed for fast execution with strong guidance: you review the plan first, then run build with clear priorities.

## When to use this
- First-time LaunchPulse projects
- MVPs with clear scope
- Fast time-to-first-version

## How it works
1. Choose Web App or Mobile App.
2. Select Autonomous AI Software Engineer.
3. Review generated plan + feature list.
4. Reorder/remove features before approval.
5. Approve and run the build.

## What you can do with it
- Generate an end-to-end first version
- Iterate with focused change requests
- Move directly into deployment workflows

## Example use cases
- Founder validating a market in 2 weeks
- Agency delivering client MVP quickly

## Example prompt
"Build a web app for HR consultants to manage candidate pipelines and client roles."
