# Autonomous AI Software Engineer

## Overview
This is the default build mode in LaunchPulse. One agent handles planning and execution from start to finish.

## When to use it
- First project
- Small-to-medium MVPs
- You want speed with minimal setup

## How it works
- You choose Web App or Mobile App
- LaunchPulse creates a build plan and feature list
- You review and edit priorities
- After approval, the build runs end-to-end

## Example use cases
- Founder building first SaaS MVP
- Agency building a client v1 quickly
