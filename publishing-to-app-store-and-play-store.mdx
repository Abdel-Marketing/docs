# Publishing to App Store & Play Store

## Overview
LaunchPulse supports app store publishing workflows for mobile projects.

## Before you start
- App store accounts and required credentials
- App identifiers/package names ready
- A mobile project that is ready for release testing

## Step-by-step guide

<Steps>
  <Step title="1) Open publish workflow">
    Start publishing from your mobile project.
  </Step>
  <Step title="2) Choose targets">
    Select iOS, Android, or both.
  </Step>
  <Step title="3) Enter required store details">
    Provide the requested account and app configuration fields.
  </Step>
  <Step title="4) Start publish">
    LaunchPulse runs build and submission workflow for selected target(s).
  </Step>
</Steps>

## What happens next
- Monitor submission status
- Resolve any store-side validation issues
