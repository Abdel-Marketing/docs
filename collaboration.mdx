# Collaboration

## Overview
Collaboration lets multiple people work on the same project without duplicating work.

## When to use it
- Co-founder or teammate support
- Review-only stakeholders

## How it works
Invite members and choose a permission level:
- Viewer: can review
- Editor: can make changes

## Example use cases
- Founder + developer handoff
- Mentor review access
