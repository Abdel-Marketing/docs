# Projects Dashboard

## Overview
Projects Dashboard is where your apps live. It helps you create new builds, reopen active projects, and keep momentum across ideas.

## When to use this
- Starting a new app
- Returning to an in-progress build
- Reviewing project status before iteration

## How it works
Use the sidebar projects view to open, create, or switch projects.

## What you can do with it
- Start web/mobile projects
- Resume prior builds quickly
- Organize your work by project

## Example use cases
- Keep client projects separate from internal tools
- Resume paused MVP work without losing context
