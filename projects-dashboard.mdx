# Projects Dashboard

## Overview
The Projects Dashboard is where every LaunchPulse app lives.

## When to use it
- Start a new app
- Resume an existing project
- Switch between projects

## How it works
Use the sidebar file icon to open your projects list. From there you can open, create, and review project status.

## Example use cases
- Start a second product idea without losing your first one
- Hand off a project to a teammate with Collaboration
