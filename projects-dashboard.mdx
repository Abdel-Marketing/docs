# Projects Dashboard

Projects Dashboard is where your product work begins and where momentum is maintained across every iteration.

## When to use this
- Create a new product
- Resume active builds
- Track progress across multiple ideas

## How it works
- Open projects list from the sidebar
- Create/open projects and review status
- Jump into the next highest-priority task

## What you can do with it
- Keep MVPs organised
- Avoid context loss between sessions
- Run parallel product experiments
