# Web App Development

Use this path when your fastest route to value is a URL users can access immediately.

## When to use this
- SaaS products
- Internal dashboards and tools
- Admin and operations portals

## How it works
- Choose Web App during setup
- Build core flows with LaunchPulse agent workflows
- Deploy and optionally connect custom domain

## What you can do with it
- Launch MVPs fast
- Add auth, storage, AI, and payments
- Iterate continuously after launch

## Example use cases
- B2B analytics SaaS MVP
- Support ticket internal tool
