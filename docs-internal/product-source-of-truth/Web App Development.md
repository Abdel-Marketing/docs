## Overview

LaunchPulse V2 supports building **full web applications** — not just static sites. Describe what you want, and LaunchPulse generates a complete web app with frontend, backend, database, auth, and deployment ready to go.

---

## What you can build

- SaaS dashboards
- Marketplaces and directories
- Internal tools
- Content sites with auth + payments
- AI-powered web apps (chat, search, generation)

---

## How it fits with the rest of LaunchPulse

| Layer | Web-specific behaviour |
| --- | --- |
| Build engine | Autonomous / Multi-Agent mode shapes a web architecture |
| Payments | Stripe for subscriptions, one-time payments, paywalls |
| Deployment | Hosted deployment + custom domain attachment |
| AI services | Built-in, no keys to manage |
| Storage / DB | Managed cloud storage and database |
| Auth | Built-in authentication |

---

## Deployment

Web apps are deployed to LaunchPulse hosting by default. You can attach a **custom domain** to take it live under your own URL. (See the Custom Domain page.)
