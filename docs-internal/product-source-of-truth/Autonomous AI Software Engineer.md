## Overview

The Autonomous AI Software Engineer mode is designed to feel like you hired a full-stack engineer who:

- plans the product
- builds it end-to-end
- connects the services you need
- checks quality as it goes

It's the default build mode in LaunchPulse V2 for users who want a single, capable agent driving the whole project.

---

## How it works

### 1) Choose what you're building
Pick **Mobile App** or **Web App**. This shapes the architecture, UI patterns, and feature set.

### 2) Choose the build mode
Select **Autonomous AI Software Engineer**.

### 3) Click "Create" → LaunchPulse builds a real plan
Instead of jumping into random screens, LaunchPulse produces:
- a structured **build plan**
- a clear **feature list** (your initial backlog)
- priorities and dependencies (what must be built first)

### 4) You stay in control before building starts
- **Remove features** you don't want yet
- **Reorder priorities** (MVP-first)
- **Add context** (target users, competitors, constraints, tone, pricing)
- **Steer the build** with extra instructions

### 5) Build on autopilot, end-to-end
After approval, the agent builds the product front-to-back with live preview and visible code.
