## Overview

LaunchPulse V2 supports building **native mobile apps** end-to-end. You can describe a mobile product in plain language, let LaunchPulse build it, and then deploy directly to the **App Store** and **Google Play Store** without leaving the platform.

RevenueCat is integrated for mobile subscription and in-app purchase flows, so monetisation is ready from day one.

---

## What you can build

- Consumer apps (social, content, lifestyle, productivity)
- Subscription apps (with paywalls + RevenueCat)
- AI-powered mobile experiences (chat, summarisation, recommendations)
- Apps with native auth, storage, and payments wired in

---

## How it fits with the rest of LaunchPulse

| Layer | Mobile-specific behaviour |
| --- | --- |
| Build engine | Same Autonomous / Multi-Agent flow, with mobile-shaped architecture |
| Payments | RevenueCat for App Store / Play Store IAP and subscriptions |
| Deployment | Direct publish to App Store Connect + Google Play Console |
| AI services | Same built-in AI services, no API keys needed |
| Storage / DB | Same managed storage layer |
