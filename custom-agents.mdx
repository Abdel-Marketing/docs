# Custom Agents

## Overview
Custom Agents let you save role-specific helpers for repeated work.

## When to use it
- You keep repeating the same instructions
- You want a dedicated reviewer or QA helper

## How it works
Create an agent with a name, short description, prompt, model setting, and tools. Keep tool access minimal for safety and predictability.

## Example use cases
- Code Reviewer agent
- QA Tester agent
- Product Manager agent
