# Troubleshooting

When velocity drops, this guide helps you identify the likely cause quickly and recover without guesswork.

## When to use this
- Build appears stuck
- Billing/credits block progress
- Deploy or publish flow fails

## How it works
1. Reproduce issue clearly
2. Match to issue category
3. Apply one fix at a time
4. Escalate with context if still blocked

## Common issues

| Issue | Likely cause | First fix |
| --- | --- | --- |
| Build stuck | Credits depleted or interrupted run | Check credits and rerun from latest approved step |
| Preview stale | Build in progress or failed task | Review build status then refresh preview |
| Domain not live | DNS mismatch/propagation | Verify records and wait propagation window |

## Support handoff checklist
- Project ID
- Exact steps to reproduce
- Expected vs actual behaviour
- Timestamp and affected environment
