# AI Services

## Overview
LaunchPulse includes built-in AI services so you can add AI features without setting up separate provider accounts.

## When to use it
- Chat features
- Summaries and extraction
- AI-powered search/recommendations

## How it works
You request AI features in plain language. LaunchPulse integrates the required app logic and service connections.

## Example use cases
- In-app assistant
- Document summary workflow
- Smart content suggestions
