# Agents

## Overview
LaunchPulse supports different agent workflows depending on your project size and style.

## Comparison

| Option | Best for | What it does | Status |
| --- | --- | --- | --- |
| Autonomous AI Software Engineer | First build, fast MVP | One primary agent plans and builds end-to-end | Live |
| Feature Builder | Better planning before build | Generates a structured feature plan you can edit | Live |
| Custom Agents | Repeated specialist tasks | Reusable agents for roles like QA/review/copy | Live |
| AI Multi-Agent Mode | Larger/complex projects | Specialist agents collaborate with handoffs | Beta / in progress |
| Testing Agent | Extra QA depth | Structured test planning and validation loops | Beta / in progress |
| LaunchPulseClaw | Advanced extension use | Early hosted OpenClaw-based agent workflow | New / early |

## When to use what

- Start with **Autonomous AI Software Engineer** for your first project
- Use **Feature Builder** to tighten scope before building
- Add **Custom Agents** when you repeat the same workflows
- Explore **Multi-Agent Mode** and **Testing Agent** for larger builds
