# Agents

LaunchPulse is built around a product-led build system: plan first, build with context, then validate before launch. This page helps you pick the right agent workflow for your current stage.

## How the LaunchPulse build system works

<Steps>
  <Step title="Plan">
    Your prompt is converted into a structured plan and feature list.
  </Step>
  <Step title="Build">
    Agents implement the app based on approved priorities.
  </Step>
  <Step title="Review and test">
    Validation and testing workflows catch issues before release.
  </Step>
  <Step title="Iterate">
    You make targeted improvements with focused prompts.
  </Step>
</Steps>

## Build option comparison

| Option | Best for | How it works | Status |
| --- | --- | --- | --- |
| Autonomous AI Software Engineer | First project, speed to MVP | One primary workflow handles planning + implementation | Live |
| Feature Builder | Better scope control | Generates a feature plan you can edit before build | Live |
| Custom Agents | Repeatable specialist tasks | Reusable role-based helpers (QA, review, PM, copy) | Live |
| AI Multi-Agent Mode | Larger/complex builds | Multiple specialist agents collaborate via handoffs | Beta / in progress |
| Testing Agent | Extra QA confidence | Adds structured test planning and validation loops | Beta / in progress |
| LaunchPulseClaw | Early extension workflows | New hosted OpenClaw-based extension path | New / early |

## Which one should I use?

| If you are... | Start with |
| --- | --- |
| Shipping your first product | Autonomous AI Software Engineer |
| Unsure about scope | Feature Builder |
| Repeating the same review steps | Custom Agents |
| Working on a larger system | AI Multi-Agent Mode |
| Tightening release quality | Testing Agent |
| Exploring advanced extensions | LaunchPulseClaw |

## Example scenarios

- **Founder with one MVP idea:** Autonomous + Feature Builder
- **Team with recurring QA needs:** Autonomous + Custom QA Agent + Testing Agent
- **Complex product with many dependencies:** Feature Builder + Multi-Agent Mode
