# What is LaunchPulse?

LaunchPulse is an AI-powered software creation platform for turning ideas into real products.
Describe what you want to build, choose whether it should be a web app or mobile app, preview the result as it takes shape, and publish when it is ready.

From one workspace, you can build dashboards, games, internal tools, SaaS MVPs, workflow apps, and more—without needing a large engineering team to get started.

<Tip>
  New to LaunchPulse? Start with one clear use case and one core workflow, then expand after your first usable version is live.
</Tip>

## How LaunchPulse works

<Steps>
  <Step title="Start with an idea or choose one from the AI app catalog">
    Begin from your own concept or pick a validated direction from LaunchPulse's AI app idea discovery area.
  </Step>
  <Step title="Describe what you want to build">
    Write a clear prompt that defines users, outcomes, and the main features for version one.
  </Step>
  <Step title="Review and refine the plan">
    Inspect the proposed scope, adjust priorities, and keep your first release intentionally focused.
  </Step>
  <Step title="Preview the product live">
    Watch your app take shape and validate the core flow before expanding functionality.
  </Step>
  <Step title="Iterate with prompts">
    Improve UX, add capabilities, and tighten details in fast, AI-assisted cycles.
  </Step>
  <Step title="Deploy or publish">
    Deploy web apps, connect domains, and publish mobile apps when your product is ready.
  </Step>
</Steps>

## Example first prompt

```text
Build a mobile app for fitness coaches to manage clients, track workouts, send reminders, and accept monthly subscriptions. Start with a clean dashboard, client profiles, workout plans, and a simple booking flow.
```

## Good first project checklist

- [ ] Pick one clear user
- [ ] Define the main problem
- [ ] Choose web or mobile
- [ ] List 3–5 must-have features
- [ ] Keep version one simple
- [ ] Preview and test the main flow before adding more
