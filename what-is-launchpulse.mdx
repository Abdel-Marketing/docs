# What is LaunchPulse?

LaunchPulse is a product-building workspace designed for speed, clarity, and shipping momentum. You describe what to build, shape the plan, and move to a working app without stitching together a stack of disconnected tools.

## What you can build

- SaaS MVPs with authentication, storage, and billing
- Internal tools for operations, support, or analytics
- Mobile MVPs with app-store publishing workflows
- AI-enabled products that need practical iteration speed

## How LaunchPulse works

<Steps>
  <Step title="Describe the product outcome">
    Start with the user, problem, and core flow you need working first.
  </Step>
  <Step title="Review generated feature plan">
    LaunchPulse creates a proposed plan you can edit, trim, and prioritise.
  </Step>
  <Step title="Run the build workflow">
    Use the best-fit build mode, generate a working version, and test core journeys.
  </Step>
  <Step title="Iterate and launch">
    Improve with focused prompts, then deploy to web or publish mobile apps.
  </Step>
</Steps>
