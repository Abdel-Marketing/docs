# LaunchPulseClaw

<Warning>
New/early extension: details may change. Please verify availability with the LaunchPulse team.
</Warning>

## Overview
LaunchPulseClaw is an extension layer for hosting OpenClaw agents with LaunchPulse.

## When to use it
- You need a more standalone agent than a project-level Custom Agent
- You want hosted agent behavior tied to LaunchPulse workflows

## How it works
You prepare an OpenClaw agent definition, then host it through LaunchPulseClaw (where available).

## Example use cases
- Dedicated hosted support agent
- Reusable external-facing workflow agent
