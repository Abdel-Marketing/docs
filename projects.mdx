# Projects

A project is one app in LaunchPulse.

Each project has its own build history, preview, and iterations.

## What to do here

- Create a new app idea
- Open an existing app
- Continue work with new prompts

## Good habits

- Keep one product idea per project
- Name projects clearly
- Validate your core flow before you move on

<Info>
If you are starting from zero, follow the Quickstart first.
</Info>
