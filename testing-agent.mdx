# Testing Agent

<Warning>
Beta / in progress: this feature is still being finalised.
</Warning>

## Overview
Testing Agent is designed to add structured QA on top of normal build checks.

## When to use it
- You want stronger pre-release testing
- You need clearer bug reports and reproduction steps

## How it works
The agent can generate test plans, run checks, and report failures for follow-up fixes.

## Example use cases
- Validate onboarding before launch
- Catch regressions after major feature changes
