# Testing Agent

<Warning>Beta / in progress: details may change as the feature matures.</Warning>

Testing Agent helps you ship with higher confidence by making quality checks explicit and repeatable.

## When to use this
- Before release milestones
- After major feature changes

## How it works
- Generates test plan from product context
- Organises checks by flow and severity
- Produces actionable failure reports

## What you can do with it
- Catch regressions earlier
- Improve reproducibility of bug reports
