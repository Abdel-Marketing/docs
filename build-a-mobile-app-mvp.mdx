# Build a Mobile App MVP

## Overview
Use this guide for iOS/Android-first MVPs with a simple release path.

## When to use this
- Your users are primarily mobile
- You want store-based distribution

## How it works
<Steps>
  <Step title="Create mobile project">Choose Mobile App when starting your project.</Step>
  <Step title="Build MVP scope">Prioritise one high-value journey first.</Step>
  <Step title="Prepare publishing">Use store publishing workflow once core flow is stable.</Step>
</Steps>

## Example prompt
"Build a mobile app for language learners with daily lessons, streak tracking, and subscription upgrade."
