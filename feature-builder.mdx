# Feature Builder

## Overview
Feature Builder creates a clear plan and prioritised feature list from your prompt.

## When to use it
- You want to avoid vague builds
- You need an MVP-first roadmap

## How it works
- LaunchPulse suggests a build plan and useful features
- You can remove, reorder, or refine before build starts
- After approval, build continues on autopilot

## Example use cases
- Trim a large idea down to a launchable v1
- Prioritise revenue-critical features first
