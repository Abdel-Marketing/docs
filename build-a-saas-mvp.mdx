# Build a SaaS MVP

## Overview
Use this guide when you want a focused SaaS launch with clear scope and fast validation.

## When to use this
- You have one customer segment in mind
- You want login, core workflow, and pricing in v1

## How it works
<Steps>
  <Step title="Define one core workflow">Pick the one user outcome your MVP must solve.</Step>
  <Step title="Use Feature Builder">Trim the feature list to essentials before build starts.</Step>
  <Step title="Build and validate">Run first build, test main flow, then iterate.</Step>
</Steps>

## Example prompt
"Build a SaaS app for freelance recruiters with lead pipeline, client tracking, and paid pro tier."
