# Build a SaaS MVP

## When to use this
When you need to validate a business problem with a working, revenue-capable product quickly.

## How it works
- Start with Quickstart
- Use Feature Builder to keep scope tight
- Build with Autonomous mode
- Add auth + payments

## Example MVP scope
- Signup/login
- One core user workflow
- Simple billing plans
