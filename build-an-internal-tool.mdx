# Build an Internal Tool

## Overview
Use this guide to build team-facing tools for operations, reporting, or process management.

## When to use this
- You need speed over polish
- A small team needs role-based access and reliable workflows

## How it works
- Start with Autonomous mode
- Keep scope to one workflow first
- Add Custom Agents for repeat QA/review checks

## Example prompt
"Build an internal ops dashboard for customer support ticket tracking with admin/reviewer roles."
