# LaunchPulse Skills

LaunchPulse Skills are reusable capability packs that help agents perform specific jobs more reliably.

Skills can include:

- workflow instructions
- domain-specific guardrails
- reusable prompts
- output formats
- tool usage guidance

## Why use Skills?

Skills help teams scale quality by giving agents pre-defined behavior for common tasks.

<CardGroup cols={2}>
  <Card title="Faster setup">
    Reuse proven configurations instead of rewriting instructions for every new project.
  </Card>

  <Card title="More consistency">
    Keep output style, safety boundaries, and workflow logic aligned across teams.
  </Card>

  <Card title="Easier onboarding">
    New builders can start with approved Skills and ship faster.
  </Card>

  <Card title="Safer operations">
    Define what agents should and should not do for each task.
  </Card>
</CardGroup>

## Common Skill types

| Skill type | What it helps with |
|---|---|
| Support Skills | Triage, FAQ handling, escalation handoffs |
| Research Skills | Summaries, comparisons, structured briefs |
| Product Skills | Spec drafting, QA checklists, release notes |
| Sales Skills | Qualification flows, discovery questions, follow-ups |
| Operations Skills | Intake workflows, classification, reporting templates |

## Suggested setup flow

<Steps>
  <Step title="Define the job">
    Start with one clear outcome the Skill should achieve.
  </Step>

  <Step title="Add constraints">
    Document boundaries, escalation rules, and prohibited actions.
  </Step>

  <Step title="Create examples">
    Add strong examples of expected inputs and outputs.
  </Step>

  <Step title="Test and iterate">
    Validate on realistic scenarios, then refine before wider use.
  </Step>
</Steps>
