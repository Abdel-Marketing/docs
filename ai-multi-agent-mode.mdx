# AI Multi-Agent Mode

<Warning>
Beta / in progress: this mode is still being completed and may change.
</Warning>

## Overview
AI Multi-Agent Mode splits work across specialist agents (planning, building, review, and QA).

## When to use it
- Larger projects with many features
- You want stronger separation between build and review

## How it works
You select Multi-Agent Mode at project creation. LaunchPulse uses agent handoffs to move work from plan to build to review.

## Example use cases
- Multi-module SaaS app with complex workflows
- Teams that want explicit review checkpoints
