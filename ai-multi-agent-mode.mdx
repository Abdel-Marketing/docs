# AI Multi-Agent Mode

<Warning>Beta / in progress: this workflow may evolve.</Warning>

For complex products, Multi-Agent Mode separates planning, build, and review responsibilities to improve quality and control.

## When to use this
- Multi-module products
- Teams that want explicit review gates

## How it works
- Work is split across specialist agents
- Handoffs move tasks through plan → build → review
- You keep decision control at checkpoints

## What you can do with it
- Scale coordination on larger builds
- Reduce single-agent bottlenecks
