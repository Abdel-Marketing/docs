# Authentication

## Overview
LaunchPulse includes built-in authentication flows as part of app builds.

## When to use it
- Login-protected apps
- User accounts with gated screens

## How it works
Ask for the auth flow you want (email/password, social, or passwordless), and LaunchPulse wires screens and logic into your app.

## Example use cases
- Member dashboard requiring login
- Admin-only pages with restricted access
